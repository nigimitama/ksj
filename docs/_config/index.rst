ksj: Kokudo Suuchi Jouhou API Client
====================================

`国土数値情報ダウンロードサービス <http://nlftp.mlit.go.jp/ksj/index.html>`_ のWeb APIを簡単に使うためのPythonパッケージです。

データ分析での利用を想定しており、本パッケージのメソッドの返り値はpandasやgeopandasのオブジェクトになります。
